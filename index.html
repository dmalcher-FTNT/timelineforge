Help & shortcuts

      

        
Workflow

        
Collect — import notes, tool exports, or reports. Refine — filter, edit, and score data quality. Deliver — pick a layout and export with preview.

      

      

        
Sharing

        
Header Share or Export → Shareable link builds a compressed URL from this site’s address () with the timeline in the hash — no server needed. Legacy links still open. Timelines that exceed the URL limit: download JSON and share the file.
